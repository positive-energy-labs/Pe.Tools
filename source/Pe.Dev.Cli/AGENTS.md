# Pe.Dev.Cli

This is an interim, automation-only source CLI. It adapts the Pe.Tools-specific
APS/Design Automation services in `Pe.Dev.RevitAutomation` to terminal commands.

Do not add web, host, build, live-loop, or source-link orchestration here. The
TypeScript host owns source web development and the SDK owns generic Revit
development/proof mechanics. Delete this project when automation gains a proven
Pea/host product surface; do not move product-specific workflow semantics into
the generic SDK.
