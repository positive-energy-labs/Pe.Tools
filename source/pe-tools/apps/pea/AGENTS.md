# Pea

## Concepts / Potential Names

| Name          | Expansion / Meaning                                        | Vibe                          |
| ------------- | ---------------------------------------------------------- | ----------------------------- |
| **Pod**       | Portable script package                                    | Workspace-native, simple      |
| **Trellis**   | Helpers hanging off PeScriptContainer that expose storage, | Automation/pod framework mode |
| **Shell**     | Outer interface around Pea                                 | CLI/TUI/operator mode         |
| **Sprout**    | Early exploratory Pea                                      | Beginner/helper mode          |
| **Bloom/Bud** | Presentation/output polishing                              | Report/doc mode               |
| **Snap**      | From snap pea; quick actions                               | Fast command mode             |
| **Pulse**     | Positive Energy signal                                     | Monitoring/audit mode         |
| **Bean**      | Cousin to Pea, smaller/friendlier                          | Lightweight mode              |
| **Seed**      | Generates pods/scripts/workflows                           | Builder mode                  |
| **Root**      | Deep model/system access                                   | Power/debug mode              |
